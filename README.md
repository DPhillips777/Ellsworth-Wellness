# Ellsworth Wellness

Ellsworth Wellness — Functional and integrative approach to health, vitality, and longevity.

Live demo

- GitHub Pages (project site): https://DPhillips777.github.io/Ellsworth-Wellness/ (enable Pages in Settings → Pages → Source: main / root)

About

This repository contains a simple static website (index.html) for Ellsworth Wellness. The site focuses on metabolic health, hormonal balance, cognitive resilience, and lifestyle-based interventions grounded in functional and integrative medicine.

Quick start (Web UI)

1. Open the repository on GitHub: https://github.com/DPhillips777/Ellsworth-Wellness
2. Click Add file → Create new file, name it `index.html` (or edit the existing file), paste your HTML, commit directly to `main`.
3. Enable GitHub Pages: Settings → Pages → Source: `main` branch, folder `/` → Save.

Quick start (Local)

1. Clone the repo:
   git clone https://github.com/DPhillips777/Ellsworth-Wellness.git
   cd Ellsworth-Wellness
2. Edit or add files locally (e.g., `index.html`, `css/`, `assets/`).
3. Commit and push:
   git add .
   git commit -m "Update site"
   git push origin main

Notes

- To disable Jekyll processing (recommended for static sites), add an empty file named `.nojekyll` at the repository root.
- The site is intentionally small and static. If you want a stylesheet, images, or a favicon, add a `css/` and `assets/` folder and reference them from `index.html`.
